# GoKwik Assignment

## Hosted on Netlify

https://gokwik.netlify.app/

## Features

- Responsive
- Local Storage API implemented for persistent state across page reloads

## Setup Dev Environment

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run start
```
